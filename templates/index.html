Exercise 3

We want to connect our Python application with a MySQL database, but we don't want to setup a database on each production server. Instead, we would like to use Docker Compose for handling this. With Docker Compose we can easily deploy the database after we've configured it, which enables us to easily deploy the system on different environments.

To complete this exercise, you will have to configure the docker compose file from guides on how to setup MySQL. You should NOT install the database on your system, but instead have docker deploy it through a docker compose file.
Please follow guides like this one on how you originally would setup a MySQL server up. Use the information to configure the Docker Compose file for connecting the Python application with the database.

You can use the existing `docker-compose.yml` file as a starting point.
